3. Please describe what is object oriented programming for nontechnical person

Object oriented programming is a kind of model or pattern that describes real-world problem using objects and interactions between them. 
It consist of classes that precise properties of created objects using methods and attributes.

Example 1:
Class is a blueprint of car, it define, that car has four wheels, engine, color and gearbox.
Object is instances of given blueprint with concrete engine power, blue color and automatic gearbox.

Class is DNA with instructions how object should looks like and do, whereas Object is real formation. (?)
